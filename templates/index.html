API Endpoints

Como Utilizar a API

Esta API possui dois endpoints principais:

  • /predict - Envia uma requisição POST com um JSON contendo informações hospitalares, e a API retornará a previsão do melhor hospital.
  • /health - Envia uma requisição GET para verificar se a API está funcionando corretamente.

Exemplo de requisição com `curl` para o endpoint /predict:

curl -X POST http://localhost:5000/predict -H "Content-Type: application/json" -d @data.json

Exemplo de requisição com `curl` para o endpoint /health:

curl http://localhost:5000/health

Você pode utilizar ferramentas como Postman ou cURL para testar os endpoints da API localmente ou em produção.
